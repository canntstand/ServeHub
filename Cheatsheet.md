# Работа с Git
Всего 2 ветки: main, dev/что-разрабатывается (во время разработки)
Варианты коммитов:
- "fix: какие-то исправления" (dev, main branch)
- "feat: добавление новых функций" (dev, main branch)
- "wip: в процессе разработки чего-то (расшифровывается work in process)" (dev branch)
- "ref: изменения в проекте без изменения функционала (к примеру строение папок и тд)" (dev, main branch)
- "new: готовый апдейт" (main branch)
